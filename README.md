# Reactive Yalc

Generic React primitives for static marketing and campaign surfaces.

This package intentionally contains no Tirrenia-specific business copy, URLs, or assets.
Consuming apps provide brand tokens, content, locale routing, and images.

## Boundaries

- Put reusable layout, campaign, action, and i18n presentation components here.
- Keep tenant/business config in the consuming site.
- Style through CSS custom properties so each brand can map its own palette.
